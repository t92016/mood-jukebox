心情點唱機 - 專案介紹

網址：https://mood-jukebox.netlify.app

============================================================
一、網站架構與技術
============================================================

(a) 用到的技術與服務

前端：HTML / CSS / JavaScript（響應式網頁，日式可愛風）
後端：Netlify Functions（安全代理，隱藏 API Key）
AI 推薦：Groq API（llama-3.3-70b-versatile），分析心情並推薦歌曲
音樂搜尋：YouTube Data API v3 + IFrame Player API（搜尋影片、嵌入播放）
資料庫：Firebase Firestore（歌曲快取 + 心情日誌）
歌詞來源：LRCLIB + lyrics.ovh（免費開源歌詞庫，自動抓取靜態歌詞）
部署：GitHub + Netlify（推 code 自動上線）

(b) 串接流程

1. 使用者輸入心情
2. 前端傳給 Netlify Function（/api/recommend）
3. Function 呼叫 Groq AI 分析心情，回傳推薦歌曲（歌名、歌手、理由）
4. 前端顯示「AI 點唱小助手」對話框，讓使用者確認或換歌
5. 使用者確認播放後，前端先查 Firestore 快取
6. 沒有快取就呼叫 YouTube API（/api/youtube-search）搜尋影片
7. 搜到後存入快取、寫入日誌、嵌入 YouTube 播放器播放
8. 同時呼叫歌詞 API（/api/lyrics）抓取歌詞顯示

============================================================
二、功能區介紹
============================================================

- 心情選擇區
- AI 點唱小助手（互動對話區）
- 播放器與歌詞
- 心情點唱牆

============================================================
三、使用者操作守則
============================================================

1. 選心情：點選上方 6 個心情按鈕，或在輸入框打字描述心情
2. 等 AI 回應：AI 會推薦一首歌，顯示在「AI 點唱小助手」對話框
3. 確認或換歌：
   - 喜歡就點「播放這首（歌手《歌名》）」
   - 不喜歡就點「換一首（AI 幫你選）」
   - 也可以打字跟 AI 說「幫我換一首」或「我改難過了」
4. 享受音樂：確認後自動播放 YouTube 影片 + 顯示歌詞
5. 看大家點什麼：往下滑到「心情點唱牆」，點別人點過的歌直接播放

============================================================
四、安全與隱私說明
============================================================

- 不需要註冊帳號，匿名使用
- 點唱牆紀錄僅保存最近 12 筆，自動覆蓋
- AI 只推薦兒童安全歌曲（歌詞健康、無成人內容）
- API Key 全部藏在後端，前端看不到
