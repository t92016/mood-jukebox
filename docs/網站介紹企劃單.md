# 心情點唱機 — 專案介紹企劃單

> 🎵 給國小高年級學生的心情音樂推薦網站
> 網址：https://mood-jukebox.netlify.app

---

## 一、網站架構與技術

### (a) 用到的服務與技術

| 層級 | 技術/服務 | 用途 |
|---|---|---|
| **前端** | HTML / CSS / 原生 JavaScript | 響應式網頁，日式可愛風 UI |
| **後端代理** | Netlify Functions | 隱藏 API Key，安全轉發 |
| **AI 推薦** | Groq API（llama-3.3-70b-versatile） | 分析心情 → 推薦歌曲（JSON mode） |
| **音樂搜尋** | YouTube Data API v3 + IFrame Player API | 搜尋影片、嵌入播放 |
| **資料庫** | Firebase Firestore | 歌曲快取（SongsCache）+ 心情日誌（MoodLogs） |
| **歌詞來源** | LRCLIB（主）+ lyrics.ovh（備援） | 免費開源歌詞庫，自動抓取靜態歌詞 |
| **部署** | GitHub → Netlify 自動部署 | 推 code 即上線 |

### (b) 串接流程（簡單條列）

1. 使用者輸入心情 → 前端傳給 **Netlify Function**（`/api/recommend`）
2. Function 帶著 **Groq API Key** → 呼叫 AI 分析心情、推薦歌曲
3. AI 回傳 JSON（歌名、歌手、理由）→ 前端顯示「AI 點唱小助手」對話框
4. 使用者確認播放 → 前端先查 **Firestore 快取** → 有就直接播
5. 沒有快取 → 呼叫 **YouTube API**（`/api/youtube-search`）搜尋影片
6. 搜到影片 → 存入快取 + 寫入心情日誌 → 嵌入 **YouTube 播放器** 播放
7. 同時呼叫 **歌詞 API**（`/api/lyrics`）→ 自動抓取歌詞顯示

## 二、功能區介紹

### 🎭 心情選擇區
- **6 個心情按鈕**：開心 / 難過 / 生氣 / 想睡 / 緊張 / 無聊
- **自由輸入框**：可以打字描述心情，例如「考試考差了，好煩」
- **瀏覽模式**：直接輸入歌手名字，瀏覽該歌手歌曲清單

### 🤖 AI 點唱小助手（互動對話區）
- AI 根據心情推薦一首歌，用聊天氣泡方式跟使用者對話
- 可以點「播放這首（歌手《歌名》）」確認播放
- 可以點「換一首（AI 幫你選）」讓 AI 重新挑選
- 也可以在輸入框自由打字對話，例如「幫我換一首」、「我現在改難過了」

### 🎬 播放器與歌詞
- YouTube 影片嵌入播放（正常 MV，非歌詞版）
- 播放控制：播放/暫停、上一首、下一首、隨機播放、音量調整
- 歌詞面板：自動抓取靜態歌詞（LRCLIB + lyrics.ovh 雙源備援）
- 歌詞可展開/收合

### 🌸 心情點唱牆
- 顯示最近 12 筆大家點的歌（最新覆蓋舊的）
- 點擊任一歌曲 → 直接在本頁播放器播放
- 匿名紀錄，無需帳號

## 三、使用者操作守則（介紹時可用）

1. **選心情**：點選上方 6 個心情按鈕，或在輸入框打字描述心情
2. **等 AI 回應**：AI 會推薦一首歌，並顯示在「AI 點唱小助手」對話框
3. **確認或換歌**：
   - 喜歡就點「播放這首（歌手《歌名》）」
   - 不喜歡就點「換一首（AI 幫你選）」
   - 也可以打字跟 AI 說「幫我換一首」或「我改難過了」
4. **享受音樂**：確認後自動播放 YouTube 影片 + 顯示歌詞
5. **看大家點什麼**：往下滑到「心情點唱牆」，點別人點過的歌直接播放

---

## 四、安全與隱私說明

- 不需要註冊帳號，匿名使用
- 點唱牆紀錄僅保存最近 12 筆，自動覆蓋
- AI 只推薦兒童安全歌曲（歌詞健康、無成人內容）
- API Key 全部藏在後端，前端看不到
